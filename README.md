# etl_pipeline
A simple ETL pipeline using pandas and scikit-learn
🔧 Tools Used
- Python
- Pandas
- scikit-learn

- ⚙️ What It Does
- **Extracts** data from `raw_data.csv`
- **Transforms** it (e.g., handles missing values, scales numerical features)
- **Loads** the processed data to `transformed_data.csv`
